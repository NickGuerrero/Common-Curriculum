Dojo Lab: Learning Path Design

Learning goal

Apply the core idea from Dojo Lab: Learning Path Design in a focused Canvas submission.

Purpose

Use AI to help design a focused learning path, then revise it using your own judgment.

Pre-Dojo Pause

Before using AI, choose one knowledge or skill gap from your inventory that matters most for moving the problem forward.

Dojo Prompt

Use this prompt with the Dojo:

I am working on this validated problem:
[paste problem]

Here is what I think I need to learn:
[paste gap inventory]

Act as Learning Coach. Help me design a focused learning path:
1. What is the smallest set of knowledge I need next?
2. What should I learn first, second, and third?
3. What can I ignore for now?
4. What sources should I use besides AI?
5. What would count as evidence that I learned enough?

Then act as Reflector. Challenge whether my plan is too broad, too vague, or disconnected from the problem.

Judgment Step

Revise the learning path after the Dojo. Keep what is useful, reject what is too broad or disconnected, and name how you will verify important claims.

Deliverable

Submit a text entry with:

  • Learning goal.
  • Sequence.
  • Sources.
  • Verification method.
  • What you are intentionally not learning.
  • How the Dojo changed the plan.

Portfolio Capture

Save the final learning path and the rejected or narrowed AI suggestions. They show how you directed learning rather than outsourcing it.
