Set Up Your Dojo

Learning goal

Stand up your AI Dojo once — a coach you’ll reuse all the way through the program. You add the Core (how it behaves), then a module for each kind of work: Resume now; Search Strategy, Interview, and Networking in the weeks ahead.

A project lets you set this up once: your instructions and the Dojo files stay loaded, your materials stay in context, and you reuse it for every piece of work. Learning to use projects is a skill worth having on its own.

What you’ll need

  • Dojo Core — download the PDF (add this to every project).
  • Resume module (this week) — download the PDF.
  • Your current resume and one real job posting you’re targeting.

Every module, always at the latest version, lives in the Dojo library.

Set it up — pick your tool

Same idea everywhere: create a project, paste the configuration (below) into its instructions, add the Dojo Core plus the module(s) you’re working on, then attach your materials.

ChatGPT Projects

  1. Left sidebar → Projects → New project.
  2. Open the project’s … menu → Project settings / Instructions and paste the configuration below.
  3. Upload the Dojo Core and this week’s Resume module to the project’s files.
  4. Start a chat and attach your resume and the job posting. (Works on free and paid accounts.)

Claude Projects

  1. Left sidebar → Projects → New project.
  2. Paste the configuration below into the project’s custom instructions.
  3. Under Files (project knowledge), click + and add the Dojo Core and the Resume module.
  4. Start a chat inside the project and attach your resume and the job posting.

Gemini Gems

  1. Open Gems → New Gem (Gem manager).
  2. Paste the configuration below into Instructions.
  3. Under Knowledge, click Add files → Upload files and add the Dojo Core and the Resume module.
  4. Open the Gem and attach your resume and the job posting. (Up to 10 knowledge files.)

Paste this as your project’s instructions

You are my Dojo -- a coach that sharpens my thinking. Follow the Dojo Core file I've added to this project: it defines how you behave (the characters and the rules). For each kind of work I add a module file -- Resume, Search Strategy, Interview, Networking. Use whichever module is in the project as your method. If more than one is loaded, ask me which I want to work on right now. The core rule: I make the decisions and do the thinking. You help me see what to change and -- once I've decided what to say -- help me write it well. You never decide for me, and you never invent anything I didn't give you. Be direct and honest: skip the cushioning and pleasantries, tell me when I'm wrong and why, challenge weak assumptions, and take initiative to suggest better approaches and save me time. To start: ask me which module I'm working on (or use the one I've added) and what I want to work on, then begin Round 1 of that module. One question at a time.

Review and personalize these instructions in any way you’d like — make them yours.

Each week, add the next module. You set the project up once. In later weeks, just download the new module — Search Strategy, Interview, Networking — from the Dojo library and drop it into this same project. The Core and your instructions stay as they are.
No project yet? You can still run it: open any chat and paste the Dojo Core and the module text (the .txt versions are easiest) as your first message. You’ll just paste them again each new chat — which is exactly why a project is worth setting up.
