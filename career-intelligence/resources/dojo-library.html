The Dojo — your context files

A Dojo is an AI project you set up once and add to as the program goes. This page always has the latest version of each file.

How it works: set up your project once (see Set Up Your Dojo), then add the Dojo Core (always) plus the module for the work you’re doing, and paste the configuration. Each module is a small file your AI uses as its method.
